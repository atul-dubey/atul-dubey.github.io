Gaufit: A Fitness tracker for cows

Summary

Livestock are expensive and since they can not talk, it becomes challenging to understand their behavior. In this project I built a device that can wrapped on necks of cows, buffalos, etc. and it is able to tracker crucial parameters like location, temperature, movement pattern, etc. I have implemented the following features in the device:

  • Location tracking using GPS
  • Livestock tagging using builtin NFC Scanner
  • MOvement and activity tracking using 6-axis motions sensor
  • Temperature measurement to understand heat cycle
  • Connected to AWS IoT for device management and data collection
